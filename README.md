# Museum of Eliminations

A (tongue-in-cheek) memorial to every elimination of the Brazilian national team in the World Cup and the Copa América since 2002. Made by AI — because the team hadn't exactly earned the effort of a human.

## Live
https://rafaeldpaula.github.io/museudaseliminacoes/

## How it works
It's a single-file static site. Open `index.html` in a browser or publish it on GitHub Pages (Settings → Pages → Deploy from a branch → main / root).

## What's inside
- A live counter of days without a World Cup elimination (reset by Norway on 2026-07-05)
- A timeline of the eliminations in the World Cup and the Copa América
- The 2019 "upset", displayed as the collection's greatest anomaly
- A roast generator, the museum shop, a candle memorial wall, and a reverse countdown to the next heartbreak
